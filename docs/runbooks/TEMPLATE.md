# Short name of the problem

**Date solved:** YYYY-MM-DD

## Symptom

What you noticed. Error messages, what was broken, how you found out.
Write it how you'd describe it to a friend — plain words are fine.

## Root cause

What was ACTUALLY wrong (not just what you restarted to make it go away).
If you never found the true cause, say so honestly — "restarting X fixed
it, cause unknown" is still useful.

## Fix

The exact commands, in order. Copy-paste them from your terminal history
(`history | tail -50`) while it's fresh. Redact any passwords/keys!

```bash
command one
command two
```

## Verify

How do you KNOW it's fixed? The command or check that proves it.

## Lesson

One or two sentences: what would past-you have needed to know to avoid
or solve this faster?
